This project aims to use basic techniques of machine learning to predict the future prices of various stocks. The specific algorithm I aim to use for this project is ‘logistic regression’, an algorithm most commonly used for classification. The classification in this specific case will be whether the a stock price will go up the day after the given date for a particular example. The features of this task will be the the values of each band in the ‘bollinger band’ technique (using a window of 20 days), on the day that the example was recorded. Python will be the designated language for this project.

Stocks

In the most abstract terms needed to build this project, a stock is like a purchasable company item that’s price can fluctuate in ways somewhat corresponding to a company's success.

Bollinger Bands

To give a brief description of what Bollinger Bands are, I’ll say that they are a tool used in the technical analysis of stocks. The bands help investors determine the state of the market in relation to the price history and depending on the how the bands are positioned, some investors can make more informed decisions about their purchases. The ‘bands’ can be thought of as lines on a graph of a stock price’s history. Usually you have an ‘upper band’ that represents a line that is 2 standard deviations above the average line for some given window of time. The ‘lower band’ is the opposite of that. It’s a line 2 standard deviations below the average price of the window. Usually investors can use where the position of the real stock price history in relation to these 3 bands to do their technical analysis with.

To download historical data about a company's stock price, visit: "http://www.google.com/finance/historical?q=[COMPANY_TICKER_SYMBOL]&output=csv" replacing the substring [COMPANY_TICKER_SYMBOL] with the ticker symbol of the desired company. For example, a link to Nvidia stock informatio would be written as: http://www.google.com/finance/historical?q=NVDA&output=csv

It takes an array containing examples for the algorithm to learn from, and another array with the classification we want the algorithm to return. I'll be using the pandas library to format the data.
