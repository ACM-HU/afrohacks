Front-End Web Development

Building a static website

I figured a good, practical introduction to front-end web development would be useful for all those new to making websites, so this project centers around just that. Front-end web development is a good skill to have under your belt as a software generalist and it will allow you to prototype ideas in a quick and visual way. In this project, I’ll detail the process of making a personal resume-type website. The technologies that will be used are HTML and CSS.

HTML/CSS/Javascript

HTML and CSS are the foundations of all websites and are commonly referred to as front-end languages, meaning it’s the part of the application that the users actually see. This can be thought of as analogous to the way an office is segmented. Although you see and interact with the people at the front-desk, there’s a lot that happens in the back to keep an office running. One of the best websites to learn HTML/CSS is here: Codecademy.com .

Project Implementation

  1. Download a text editor like Sublime Text
  2. While using sublime text for the first time, a new blank document should be opened in front of you
  3. Save the document with the extension '.html' into an easily accessible folder on your computer
  4. Type in the standard mark-up that’s included in all html files
  5. Edit the html document by adding information you would like in the webiste
  6. Add a stylesheet link to your html document, and create and save a css file in the same folder as your html file
  7. Open the html file by right-clicking, going to ‘open with’, and choosing your web browser of choice
  8. You should now be able to see your web page in your favorite browser!
